## Cách ghi câu lệnh git :
+ Làm việc với database :
  - init : KHởi tạo database
  - db : Thay đổi cơ sở d liệu
  - schema : Thay đổi cấu trúc table.
  - data : Thêm dữ liệu .
  - fix : Sữa lỗi .
  - refactor : Chuẩn hóa .
+ Làm việc với mã nguồn :
  - feat : Phát triển tính năng mới.
  - fix : Sửa lỗi.
  - UI : Giao diện .
  - refactor : Chuẩn hóa .
  - logic : Tùy chỉnh locgic code .
  - update : Cập nhật dự án .
